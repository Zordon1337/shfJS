<h2> SteamHourFarmJS - Node.js app for Farming hours on steam </h2>


<h3>How to use?</h3>


Install steam-user pkg using "npm i steam-user"<br/>
rename config.json.example to config.json<br/>
edit config.json<br/>
run index.js with "node index.js"<br/>
