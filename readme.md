<h3> SteamHourFarmJS - Node.js app for Farming hours on steam </h3>


<h5>How to use?</h5>
```
Install steam-user pkg using "npm i steam-user"
rename config.json.example to config.json
edit config.json
run index.js with "node index.js"
```
